สร้าง GitHub Token ใหม่ (ถ้าโควต้าหมด)

  1. ไปที่ github.com/settings/tokens
  2. กด Generate new token → Fine-grained tokens
  3. ตั้งชื่อใดก็ได้ เช่น csc-virtual-office
  4. ใต้ Permissions เลือก Models → Read-only
  5. คัดลอก token แล้วนำไปอัปเดต Secret ชื่อ GITHUB_MODELS_TOKEN บน Render
  6. Restart service บน Render เพื่อใช้ token ใหม่

โควต้าฟรีของ GitHub Models รีเซ็ตทุกวัน ถ้าสร้าง token ใหม่แต่ยังโดน 429 แสดงว่าโควต้ารายวันของ account นั้นหมดแล้ว ต้องรอถึงวันถัดไป หรือใช้ account GitHub คนละตัว
