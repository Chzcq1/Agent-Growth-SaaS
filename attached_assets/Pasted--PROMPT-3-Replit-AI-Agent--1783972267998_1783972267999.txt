# PROMPT (ฉบับที่ 3) สำหรับ Replit AI Agent: แก้ปัญหาแผนงานตื้นเกินไป ไม่คิดแบบมนุษย์

> **บริบท:** ทดลองใช้ระบบจริงแล้ว (ดู screenshot ที่แนบ) พบว่า Output ที่ได้ยัง**ตื้นเกินไป** เช่น สั่งว่า "สร้างโพสต์ใน Facebook เกี่ยวกับการลดลูกค้าเท โดยใช้คำว่า 'รู้หรือไม่? ร้านที่ใช้ CSC ลดลูกค้าเทได้อย่างไร?'" แล้วจบแค่นั้น — ไม่บอกว่าทำไมต้องโพสต์แบบนี้ เป้าหมายกี่คน กลุ่มเป้าหมายคือใคร ต้องโพสต์กี่กลุ่ม กลุ่มไหนบ้าง และทำอย่างไรให้คนมามีปฏิสัมพันธ์กับโพสต์จริง **ต้องแก้ทั้งระบบ Prompt ของทุก Agent ให้คิดลึกแบบคนที่มีประสบการณ์ทำการตลาดจริง ไม่ใช่แค่โยนประโยคสำเร็จรูปออกมา**

---

## 1. ปัญหาที่พบจากการทดสอบจริง (ต้องแก้ทุกข้อ)

จาก Output ตัวอย่างที่ได้:
- `สร้างโพสต์ใน Facebook เกี่ยวกับการลดลูกค้าเท โดยใช้คำว่า 'รู้หรือไม่? ร้านที่ใช้ CSC ลดลูกค้าเทได้อย่างไร?'` → **ไม่มี**: จะโพสต์กลุ่มไหน, กี่กลุ่ม, เป้าหมายยอด engagement เท่าไหร่, กลุ่มเป้าหมายคือเจ้าของร้านหรือลูกค้าปลายทาง, ควรโพสต์เวลาไหน, รูปแบบโพสต์ (ข้อความอย่างเดียว/รูป/โพล) ควรเป็นแบบไหนถึงจะมีคนคอมเมนต์
- `สร้างวิดีโอสั้นๆ สอนการตั้งค่าร้านและการใช้ TOTP` → ไม่บอกความยาว, สไตล์, ควรลงช่องทางไหน, สคริปต์คร่าวๆ
- แผนงานเป็น List คำสั่งลอยๆ ไม่มี "เหตุผลเชิงกลยุทธ์" หรือ "ผลลัพธ์ที่คาดหวัง" กำกับแต่ละข้อ

**สรุปสาเหตุ:** Agent ถูกสั่งให้ "สรุปเป็น Action Item" แต่ไม่ถูกบังคับให้ "คิดแบบนักการตลาด/นักวางแผนจริง" ก่อนสรุป ทำให้ได้ Output ที่เป็นแค่ประโยคคำสั่งลอยๆ ไม่มีน้ำหนักในการตัดสินใจ

## 2. หลักการแก้ไข: บังคับ "ชั้นคิดเชิงกลยุทธ์" ก่อนสรุป Action ทุกครั้ง

ให้ปรับ Prompt ของทุก Agent (โดยเฉพาะ Lead Hunter, Sales Assistant, Demo Agent) ให้ทำงาน 2 ชั้นเสมอ ห้ามข้ามชั้นที่ 1 ไปตอบชั้นที่ 2 ตรงๆ:

**ชั้น 1 — คิดแบบนักการตลาด (ต้องตอบในใจ/แสดงเป็น reasoning field ก่อนสรุป):**
- เป้าหมายเชิงตัวเลขของ action นี้คืออะไร (เช่น เป้าเข้าถึง 500 คน, เป้า Lead ใหม่ 5 ราย/สัปดาห์)
- กลุ่มเป้าหมายที่แท้จริงคือใคร (เจ้าของร้านทำเล็บที่มีพนักงาน 1-3 คน, อายุร้าน 1-3 ปี, มีปัญหาเรื่องคิว ฯลฯ) — ห้ามเขียนกว้างๆ ว่า "เจ้าของร้านบิวตี้"
- ทำไมต้องใช้วิธีนี้ (โพสต์/ทัก Messenger/วิดีโอ) เทียบกับวิธีอื่น เหตุผลคืออะไร
- ปัจจัยที่ทำให้คนมีปฏิสัมพันธ์จริง (hook ของโพสต์, คำถามปลายเปิดที่กระตุ้นคอมเมนต์, เวลาที่คนกลุ่มเป้าหมายออนไลน์, รูปแบบเนื้อหาที่เหมาะกับแพลตฟอร์มนั้น)
